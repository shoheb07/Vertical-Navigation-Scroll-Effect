# Vertical-Navigation-Scroll-Effect
Vertical Navigation Scroll Effect highlights the menu item in a sidebar while the user scrolls through different sections of the webpage. This is commonly called a Scroll Spy navigation.
project structure 
vertical-scroll-navigation
│
├── index.html
├── style.css
├── script.js
Features
Vertical sidebar navigation menu
Active menu highlight based on scroll position
Smooth scrolling between sections
Responsive layout
Simple and lightweight implementation
Easy to integrate into any website
Technologies Used
HTML5 – Structure of the webpage
CSS3 – Styling and layout design
JavaScript (Vanilla JS) – Scroll detection and dynamic menu highlighting
How It Works
The webpage contains multiple sections with unique IDs.
The sidebar navigation links correspond to those section IDs.
When the user scrolls the page:
JavaScript detects the current section in view.
The corresponding navigation link receives an active class.
CSS styles highlight the active navigation item.
